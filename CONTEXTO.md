# Documento de Contexto y Memoria del Proyecto
**Tecnimedical - Vitrina Digital Inteligente (MVP Etapa 1)**

---

## 📌 Visión General
Tecnimedical es una empresa venezolana (San Cristóbal, Táchira) dedicada al suministro de equipos médicos para rehabilitación y diagnóstico. Este proyecto implementa una Vitrina Digital Inteligente centrada en la captación de leads mediante un Agente de IA "Súper Agente".

### Misión del MVP
Transformar la consulta de productos médicos en una experiencia asesorada por IA, canalizando el interés del cliente directamente hacia WhatsApp para el cierre de ventas humano.

---

## 🛠️ Tech Stack (Estándar Bright Drive Labs)
- **Frontend:** React 19 + Vite + Tailwind CSS v4 + Framer Motion.
- **Backend:** Fastify (Node.js/TypeScript).
- **IA:** Google Gemini 1.5 (Pro/Flash) con Function Calling.
- **Base de Datos:** Supabase (PostgreSQL).
- **CMS:** Google Drive API (Proxy de imágenes).
- **Infraestructura:** Vercel (Frontend) + Railway (Backend).

## ⚙️ Módulos Clave
1. **Lead Gen Hero:** Captura de datos (Nombre/WhatsApp) a través de la descarga del catálogo.
2. **AI Shopping Assistant:** Gemini 1.5 entrenado para asesorar sobre productos médicos en Venezuela.
3. **WhatsApp Webhook:** Generación dinámica de enlaces de compra/cotización personalizados.
4. **Smart Product Grid:** Vitrina interactiva con pre-carga de contexto para la IA.
5. **Brand Consistency:** Integración visual 1:1 con la presencia en Redes Sociales (Instagram @tecnimedical.ve).
